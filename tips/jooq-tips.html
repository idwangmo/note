      # 一、 查看日志

      在使用Spring boot的时候，查看日志可以在bootstrap.yml中加入如下配置：

      logging:
    level:
        org.jooq.tools.LoggerListener: debug

      # 二、Java8 时间问题

      当使用JPA生成实体的时候，使用到Java8的时间，会出现Record转换不对的情况，这个时候在配置文件中加入如下配置：

      <generate>
  <javaTimeTypes>true</javaTimeTypes>
</generate>
