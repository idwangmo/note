# 添加用户到 Docker 用户组

在安装Docker之后，用如下指令将当前用户加入到用户组：

sudo usermod -aG docker $USER
1

如果提示/var/run/docker.sock权限不足，使用如下指令解决：

sudo chmod a+rw /var/run/docker.sock
1

# 配置 Docker 为国内镜像

编辑/etc/docker/daemon.json文件后，然后重启Docker服务：

{
    "registry-mirrors": ["https://registry.docker-cn.com"]
}
