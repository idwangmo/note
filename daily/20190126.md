# 2018-01-26: Git 密码管理

## Git 密码管理

在使用 Git 的时候，会遇到一些只能使用用户名和密码进行代码下载的情况，尤其是在使
用`Git Submodule`的时候，频繁的输入密码可以通过一下方式进行解决：

1. 在`home`目录创建`.netrc`文件
2. 在创建好到文件中输入并替换相关内容:

``` txt
machine example.org
    login your-username
    password your-password
```
